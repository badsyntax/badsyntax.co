This is a straightforward no BS introduction to getting started with the Vim Nerd Tree plugin.

The installation is actually really simple.

  1. Download and unzip the plugin into your ~/.vim directory
  2. Open Vim
  3. type :NERDTree or :NERDTreeToggle

I’m undecided if I’ll find this plugin useful as up till now I’ve gotten used to navigating directories on the command line. My ‘c’ and ‘d’ keyboard keys are the most worn out :) But I’m all for better productivity, so I’m gonna give it a bash.
