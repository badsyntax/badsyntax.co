Simply put, the KO3 image module does not use configuration files.

Have a look for yourself:

grep -r config PATH_TO/modules/image/

An obvious ‘expected’ configuration is to change the image driver. As you cannot do this via configuration, I will list three ways to change this:

Option 1

Change the public static property somewhere in your application.

Image::$default_driver = 'imagick';

Option 2

Pass in the driver sting as an argument for the factory method:

Image::factory($file, 'imagick');

Option 3

Extend the image class:

EG, in file: application/classes/image.php

abstract class Image extends Kohana_Image {

  public static $default_driver = 'imagick';

}
