I'm happy to announce that I've released a BETA version of the new javascript spellchecker I've been working on.

The jQuery Spellchecker is a Javascript spellchecker in the form of a lightweight jQuery plugin that can be used to check the spelling of text within a form field or within a DOM tree. This new spellchecker is a huge improvement on the orginal spellchecker plugin I wrote, and comes with many new features:

Features:

  • Unicode support for working with different languages
  • Excellent HTML parsing for working with text in a DOM tree
  • Multiple fields support

Demos

  • DOM tree demo
  • Textarea demo
  • Languages demo
  • Multiple fields demo

WYSIWYG demos

  • Redactor demo
  • Bootstrap-wysihtml5 demo

I've developing the plugin on github under a MIT Open Source license. Feel free to fork and contribute!

The documentation is a little lacking but I will gradually add to it over time. Please post any issue you find in the issue tracker.
