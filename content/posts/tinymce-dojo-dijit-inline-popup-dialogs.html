Following on from my original post on how to create a custom TinyMCE inline popups plugin (using jQuery UI), I have now created the Dojo Dijit version.

This plugin is directed at those who are already using Dijit as their UI library, and is intended to bring consistency to the interface. I have decided not to code the custom alert and confirm dialogs (as is with the original TinyMCE inline popups plugin), as I don’t see a real need to replace these default native system dialogs.

You can check out the project on github, or take a look at the demo.

Installation is straightforward:

  1. Ensure you have loaded the dojo library and theme
  2. Ensure you have loaded the dialog dijit
  3. Copy the ‘dojoinlinepopups’ folder into the tinymce plugins directory
  4. Disable the ‘inlinepopups’ plugin, and enable the ‘dojoinlinepopups’ plugin when initiating tinymce.
  5. That is all! If you get stuck, refer to the demo!
