I’m a fan of code formatting tools, and of Sublime Text. I wanted to find a plugin that could ‘beautify’ my Sass files from within the text editor. After a bit of googling around, I could not find such a plugin, so I decided to create one myself.

Introducing SassBeautify - A Sublime Text 2 plugin that beautifies Sass files: https://github.com/badsyntax/SassBeautify

It’s a pretty basic plugin, and is essentially a wrapper around the ‘sass-convert’ utility. It’s also my first bit of python code. Although I never coded in python before, I found the concepts and syntax familiar. The hardest part was getting familiar with the Sublime Text API, but I learned a lot from looking at the code within the ‘Default’ package.

If you would like to use the plugin, you can install it from the sublime text package manager, and it should would on GNU/Linux; Windows and OSX. (Read the README for more detailed installation instructions.) If you find it doesn’t work as you expect, or you would like to make some suggestions, please create a ticket in the issue tracker.
