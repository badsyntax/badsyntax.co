This is an brief introduction on how to get started with the nodejs memcached module.

The instructions below assume you're working on Debian based operating system.

Install and configure packages

Install Memcached: sudo apt-get install memcached

Configure memcached: sudo vi /etc/memcached.conf

Start the memcached server: sudo /etc/init.d/memcached restart

Install the nodejs module: npm install memcached

Sample basic usage

The memcached API is asynchronous, you will need to handle returned values with callback functions.

var Memcached = require('memcached');

var memcached = new Memcached('localhost:11211');

var lifetime = 86400; //24hrs

memcached.set('hello', 'world', lifetime, function( err, result ){
  if( err ) console.error( err );
  console.dir( result );
});

memcached.get('hello', function( err, result ){
  if( err ) console.error( err );
  console.dir( result );
});

You can view many more examples in the nodejs memcached git repository.
