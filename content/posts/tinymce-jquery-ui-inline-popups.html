This post explains how to create a custom TinyMCE inline popups plugin using the jQuery UI dialog widget.

Background

If you’re using the jQuery UI dialog in your interface, it may seem odd when a different dialog is displayed in TinyMCE. This plugin can help bring consistency to your interface by using the jQuery UI dialogs for TinyMCE popups.

Installing the plugin

  1. Create a new folder called ‘jqueryinlinepopups’ in the tinymce plugins directory.
  2. Download editor_plugin.js file, and place it within the ‘jqueryinlinepopups’ folder.
  3. Enable the ‘jqueryinlinepopups’ plugin. You set this in the config object to pass into TinyMCE.init().

Example TinyMCE init: 

tinyMCE.init({

    // General options
    mode : "textareas",
    theme : "advanced",

    // Plugins
    plugins :  'jqueryinlinepopups',

    // Theme options
    theme_advanced_buttons1 : 'formatselect,|,bold,italic,|,link,unlink,|,image,media,|,code,help,fullscreen',
    theme_advanced_buttons2 : "",
    theme_advanced_buttons3 : "",
    theme_advanced_buttons4 : "",
    theme_advanced_resize_horizontal : false,
    theme_advanced_toolbar_location : "top",
    theme_advanced_toolbar_align : "left",
    theme_advanced_statusbar_location : "bottom",
    theme_advanced_resizing : true,

    // Skin options
    skin : "cirkuit"

});

Bugs

I’ve tested this plugin with TinyMCE versions 3.3.9.2 & 3.4.2, and it works in IE(6,7,8,9), Chrome, Firefox & Opera, but if you do notice any bugs, please let me know in the issue tracker.
